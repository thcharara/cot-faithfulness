  ## Tier 2: Corpus Sample (10)

  One trace per task-set combination (4 tasks x 2 sets = 8 slots) plus 2
  additional traces selected for edge-case diversity. Traces were chosen at
  mid-range sentence counts within each slot to represent typical corpus members,
  with the two extras capturing the extremes of the length distribution.

  (full corpus range: 28-558). "Completed" means the trace contains a RULE
  sentence (the model stated a final answer); "truncated" means generation hit the
  token limit while still actively reasoning.

  ## Regenerating the Full Corpus

  The full 320-trace coded corpus can be regenerated by running the Study 1
  pipeline:

  ```bash
  # Requires: ANTHROPIC_API_KEY environment variable, ~$2.50 API cost
  python study1_corpus/scripts/study1_generate_traces.py
  python study1_corpus/scripts/study1_segment_traces.py
  python study1_corpus/scripts/study1_preprocess_traces.py
  python study1_corpus/scripts/study1_batch_code_traces.py submit --traces all
  python study1_corpus/scripts/study1_batch_code_traces.py poll
  python study1_corpus/scripts/study1_batch_code_traces.py download
```

  Note: Trace generation requires a GPU and the DeepSeek-R1-Distill-Llama-8B model
  (see download_model.py). Auto-coding requires an API key.
